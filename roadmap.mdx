Below are the upcoming features that we are working on:

<Steps>
    <Step title='Data API (Released)'>
        Providing a set of base API endpoints for community users and developers
        to quickly access data and information about the Eigen Layer ecosystem.
    </Step>
    <Step title='Individual AVS Pages'>
        Get a detailed look of each AVS including their historical data,
        operators, stakers, strategy distributions, and risks.
    </Step>
    <Step title='AVS Risk Framework'>
        Aggregate and construct a framework which consists of specific risks of
        slashing, distribution of rewards, methods of data verification, and
        staker and operator data for an intuitive understanding of the risks
        involved operating and staking into an AVS.
    </Step>
</Steps>
